# 開発ログ出力コマンド

このコマンドは、現在のセッションの会話内容を開発ログとしてマークダウンファイルに保存します。

## 実行手順

1. **設定ファイルの読み込み**: `/home/claude-code/.claude/log-config.json` を読み込み、出力先フォルダを取得

2. **タイトルの決定**: 今回のセッションで行った主な作業内容を簡潔に表すタイトルを決定。特に重要なものや大きな変更を優先する。（例：「ランダム報酬の実装」「バグ修正」など）

3. **ファイル名の生成**:
   - Bashで `date +"%y%m%d_%H%M"` を実行して現在日時を取得
   - `YYMMDD_HHmm タイトル.md` 形式でファイル名を作成

4. **ログ内容の作成**: 会話履歴を参照し、以下のフォーマットでマークダウンを作成：

```markdown
# [タイトル]

**日時**: YYYY年MM月DD日 HHmm
**プロジェクト**: [関連プロジェクト名]

## 概要
[今回のセッションで何を行ったかの簡潔な説明]

## 実施内容
[具体的に行った作業のリスト]
- 作業1
- 作業2

## 変更ファイル
[変更・作成したファイルのリスト（あれば）]

## 技術的なポイント
[実装で使った技術や注意点など]

## 実装時に生じた問題とその解決方法
[実装時に生じた技術的な問題とそれをどうやって解決したか。次回以降同じトラブルを回避するための方法やアドバイス(あれば)]

## 次回のTODO
[残タスクや次に取り組むべきこと]

## メモ
[その他気づいたことや参考情報]
```

5. **ファイルの保存**: Write ツールで設定ファイルの `outputFolder` に保存

6. **完了報告**: 保存先パスとファイル名をユーザーに報告

## 注意事項
- 出力先フォルダを変更したい場合は `/home/claude-code/.claude/log-config.json` の `outputFolder` を編集してください
- Windows形式のパスは `/mnt/c/...` 形式で指定してください
