# データタブ（カレンダービュー）実装計画

## Context

ページ画面の3タブのうち「データ」タブが `TabPlaceholder` で空状態。penデザイン `Screen - Calendar Data` およびスクリーンショットに従い、月間カレンダービューを実装する。ルーティンのチェック履歴をドット表示し、マイルストーン達成日を星マーカーで可視化する。

既存の日付ユーティリティ（`getMonthDates`, `getDayLabel`, `getTodayString`）がすべて揃っているため、UIコンポーネント新規作成 + ページ画面統合のみ。
