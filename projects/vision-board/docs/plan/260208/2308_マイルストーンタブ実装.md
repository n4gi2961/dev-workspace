# マイルストーンタブ実装計画

## Context

ページ画面の「マイルストーン」タブが `TabPlaceholder` で空状態。penデザイン（`Screen - Milestone`）に従い、マイルストーンリストUIを実装する。バックエンド（`usePages.saveMilestones`）は完全実装済みのため、UIコンポーネント作成 + ページ画面への統合のみ。

---

## penデザイン仕様

```
目標達成までの中長期的なステップを管理。

⠿ ✓ ハーフマラソン走る          2025/1/9    ← 完了: 緑チェック + 日付
⠿ ✓ マラソンにエントリーする    2026/1/10   ← 完了: 緑チェック + 日付
⠿ □ 月1 100km走る                          ← 未完了: 灰色四角
⠿ □                                         ← 空スロット

  新しいマイルストーンを追加...         ＋    ← 追加行
```

- 行高さ: 56px、角丸: 8px、padding: `[12, 16]`、gap（行内）: 12px
