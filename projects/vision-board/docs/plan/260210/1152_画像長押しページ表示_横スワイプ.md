# 閲覧モード: 画像長押しでページ表示 + 横スワイプ移動

## Context

閲覧モード（全画面含む）から画像ノードを長押しすると、既存のページ詳細画面を開く。
ページ詳細画面自体に水平PagerView + 横ドットインジケーターを追加し、左右スワイプで他ページに移動できるようにする。
ノードは左上座標で「上にあるもの＞左にあるもの」の優先度で並び替え。
新規ルート不要 — 既存の `page/[nodeId]` 画面を内部改修するだけ。

## ステップ一覧

### Step 1: PageContent コンポーネント抽出
**新規ファイル**: `components/page/PageContent.tsx`

既存 `page/[nodeId].tsx` のUI・ロジック（SelectorDropdown, HeaderActions, スクロール処理, タブ, 全hooks）をそのまま抽出。

### Step 2: page/[nodeId].tsx を PagerView ラッパーに改修
- 画像ノードを位置ソート（y昇順 → x昇順）
- PagerView で PageContent をラップ
- HorizontalPageIndicator（FocusScreenの縦ドットの横版）
- ノード1つ以下/テキストの場合は従来通り単一表示

### Step 3: CanvasNode に onLongPress 追加
viewモードのPressable に onLongPress ハンドラ追加

### Step 4: BoardCanvas props受け渡し
onLongPressNode prop追加、CanvasNode に渡す

### Step 5: index.tsx — 長押しハンドラ
`router.push('/(main)/page/${nodeId}')` で既存ルートに遷移

## 変更ファイル一覧

| 順序 | ファイル | 変更 |
|------|---------|------|
| 1 | `components/page/PageContent.tsx` | **新規** — 既存ページUIの抽出 |
| 2 | `app/(main)/page/[nodeId].tsx` | PagerView ラッパー + HorizontalPageIndicator |
| 3 | `components/canvas/CanvasNode.tsx` | `onLongPress` prop追加 |
| 4 | `components/canvas/BoardCanvas.tsx` | `onLongPressNode` prop追加・受け渡し |
| 5 | `app/(main)/(tabs)/index.tsx` | 長押しハンドラ + ルーター遷移 |
