     実施タスク一覧

     Task 1: Home画面の再設計（FAB展開メニュー + 全画面ボタン）

     1-1. FAB展開メニュー（Apple Freeform参考）

     - 右下の既存FABを「+」ボタン（汎用プラス）に変更
     - タップすると上方向にメニューが展開
     - メニュー項目（アイコン + ラベル）:
       - 画像追加（image-plus アイコン）
       - テキスト追加（type アイコン）
       - 図形追加（shapes アイコン）
       - メモ帳追加（sticky-note アイコン）
     - 展開時: FABの「+」が「x」（閉じる）に変化
     - 背景にオーバーレイ（半透明暗色）

     1-2. Screen - Home (通常状態)

     - 現在のHome (Canvas) SxQqu を更新
     - 変更点:
       - 右下FABのアイコンを plus に変更（現在は image-plus）
       - 左下に全画面ボタンを追加（expand/maximize アイコン、控えめなデザイン）

     1-3. Screen - Home (FAB展開状態)

     - 通常状態をコピーして展開メニューを上にオーバーレイ
     - FABの上に4つのメニューアイテムが縦に展開
     - 背景に半透明オーバーレイ

     1-4. Screen - Home (Fullscreen Mode)

     - 全画面ボタン押下後の状態
       - TopBar、Tab Bar、FAB、全画面ボタン すべて非表示
       - 画面全体がキャンバスのみ
       - 「ダブルタップで解除」のトースト表示（画面中央下部、2秒で消える想定）

     1-5. Screen - Home (ノード選択状態 - Edit)

     - ノードをタップした時の状態
       - 選択中のノードにドラッグハンドル/リサイズハンドル表示
       - ノード周囲に選択枠線

     ---
     Task 2: Focus画面への縦インジケーター統合

     現状の問題

     - 縦ドットインジケーター（vbbdz）が存在するがどの画面にも統合されていない
     - Focus画面は verticalレイアウトで、インジケーターを横に並べる方法が不明だった

     解決策

     既存のFocus画面（Q6l4R）の構造を変更:
     Screen - Focus (vertical)
       ├── StatusBar
       ├── Content Wrapper (horizontal, fill_container) ← 新規追加
       │   ├── VisionBoardCard area (vertical, fill_container)
       │   └── Vertical Indicator (vertical, centered) ← vbbdzの中身をここに配置
       └── Tab Bar

     - Content Wrapperをhorizontalレイアウトで新規追加
     - 既存のfocusContentとインジケーターを横並びにする
     - インジケーターは画面右端に配置（padding-right: 8程度）

     Task 3: リップルエフェクトの見た目設計

     - Focus画面上で「タップ地点から広がるリップル」の視覚スタイルを定義
     - 設計内容:
       - リップル円の色/透明度（放射状グラデーション）
       - リップル円の境界線スタイル（ぼかし具合）
       - サイズ感（広がった状態の円1つ）
     - VisionBoardCard上にオーバーレイとして1枚の静止画を設計
     - アニメーション自体はReanimated + Skiaで実装するため、あくまでスタイル定義のみ

     ---
     Task 4: 画像形状選択コンポーネント

     - ノード追加時の画像形状を選ぶボトムシートまたはパネル
     - 選択肢: FREE / ORIGINAL / SQUARE / LANDSCAPE / PORTRAIT
     - 各形状をアイコン（アスペクト比を示す矩形アイコン） + ラベルで表現
     - 選択中の状態（ハイライト）

     ---
     Task 5: テキストノード編集パネル

     - テキストノードのスタイルを編集するボトムシート/パネル
     - 操作項目:
       - フォントサイズ（S / M / L または数値スライダー）
       - テキストカラー（プリセットカラーパレット）
       - フォントウェイト（Regular / Bold）
     - 既存のデザイン言語（ダーク背景、#1F1F1F系）と統一
