# Focus Screen 実装計画

## Context

ボトムバーの「Focus」タブ（現在 `ambient.tsx` でプレースホルダー状態）に、ボード内の画像を全画面で没入的に閲覧する機能を実装する。Vision Board アプリの核心機能の1つで、ユーザーが目標のビジュアルに集中するための画面。

---

## 仕様まとめ

- Focusタブを押すと、**現在のボードの画像ノードをランダム順で全画面表示**
- **シャッフルはタブを開くたびに実行**
- **縦スワイプ**で画像を切り替え（PagerView）
- 画像は**なるべく大きく**、ボード上の**使用アスペクト比**（`node.width / node.height`）で表示
- 背景は**同一画像のブラー版が画面全体を覆う**（余白なし）
- 右端に**縦ドットインジケーター**（画像にオーバーレイ、追加スペースを取らない）
